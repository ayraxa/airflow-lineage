# Airflow Lineage

## Overview
Airflow Lineage is a tool designed to enhance data lineage tracking within Apache Airflow. It provides visibility into data workflows, making it easier to understand data flow and dependencies.

## Features
- Visualize data lineage
- Track data dependencies
- Integrate with existing Airflow DAGs

## Run
1. Create a `.env` file
2. Run `docker compose up`
3. Open Airflow, Marquez, and Streamlit in the browser
